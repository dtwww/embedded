MATRIX IV简介：

    MATRIX IV 是由广州友善之臂电子公司设计生产的一款ARM9参考板，它基于三星公司的ARM处理器S3C2410X， 采用6层板设计。S3C2410X 使用 ARM920T 核，内部带有全性能的 MMU( 内存处理单元 ) ，它适用于设计移动手持设备类产品，具有高性能、低功耗、接口丰富和体积小能优良特性。 MATRIX IV 正是基于此芯片本身的各种特点而设计的。

    MATRIX IV 在尽可能小的板面上集成了网卡、音频、 USB Host 、 USB Slave 、一个标准串口、键盘和一些用户灯等设备接口，并且引出 CPU 的大部分信号引脚，正如你所看到的，它是一个完全可以独立运行的电路板。我们的目的：让它可不仅以作为专业研发工程师的参考板/试验板，进行项目或产品的原型设计；更可以作为一台移动微型电脑的主板给嵌入式DIY爱好者直接使用。

    在软件上，我们采用了韩国MIZI公司所公布的开放源代码的免费嵌入式操作系统arm-Linux，基于该平台及其开发工具包，我们编写和移植了丰富的软件供用户使用和参考，如控制台模式下的Mp3 播放器，Web 服务器(支持CGI)， Ftp及其服务器，Telnet及其服务器，键盘标准驱动程序等等。这些软件的二进制可执行程序及其源代码均收录在所附带的光盘中，并预装在 MATRIX IV 参考板上，用户开机即可使用。

    为让用户能够更加方便的使用MATRIX IX，我们编写了十分详细的使用手册，它包含在MATRIX IV套件中，另外我们会根据用户的要求和反馈意见，陆续在网站上推出一些使用MATRIX IV实例的文章给大家。

    MATRIX IV分为两种版本面向用户：个人版和专业版。二者的软硬件完全相同，不同之处在于所提供的电路图格式和售后服务方式。个人版将提供pdf格式的电路原理图和Email方式的服务支持，专业版将提供Protel格式的电路原理图和电话方式的服务支持。
